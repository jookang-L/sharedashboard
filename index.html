    즐겨찾기 링크 편집

    이름과 주소를 입력하세요. 한 번 클릭은 새 탭으로 이동, 더블클릭은 이 창이 열립니다.

    Google Sheets API 키 발급 안내

    1. API 키 발급 단계

    1. Google Cloud 콘솔에 접속하여 프로젝트를 선택하거나 생성합니다.
    2. 좌측 메뉴 API 및 서비스 > 라이브러리에서 "Google Sheets API"를 검색해 [사용 설정]을 클릭합니다.
    3. 좌측 메뉴의 API 및 서비스 > 사용자 인증 정보 탭으로 이동합니다.
    4. 상단의 [+ 사용자 인증 정보 만들기] 버튼을 누르고 [API 키]를 선택합니다.
    5. 생성된 API 키를 복사하여 저장합니다. 보안을 위해 키 제한 설정을 통해 Google Sheets API로만 사용 범위를 제한하는 것이 좋습니다.

    2. 사용 시 주의사항

    시트 공개 설정: 서비스 계정은 특정 이메일에 권한을 줄 수 있지만, API 키 방식은 시트 자체의 공유 설정이 "링크가 있는 모든 사용자에게 공개"되어 있어야 데이터를 읽어올 수 있습니다.

    스프레드시트 유의사항

    대시보드는 아래 시트 탭 이름을 사용합니다. 새 스프레드시트를 만들 때 탭 이름을 맞춰 주세요.

    • 시트1 — 시간표(시트에서 불러올 때)
    • 할일 — 할 일 목록
    • 메모 — 메모
    • 링크 — 즐겨찾기 링크

    Apps Script 유의사항

    앱스크립트는 사용자가 대시보드에 적은 할일·메모를 구글 시트에 반영하고 싶을 때 웹앱 URL을 넣습니다. 필요 없다면 비워 두고 넘어가도 됩니다.

    대시보드에 연동한 그 스프레드시트를 연 뒤, 반드시 그 파일에서 확장 프로그램 → Apps Script를 여세요. 스크립트만 따로 만든 빈 프로젝트에 붙여 넣으면 getActiveSpreadsheet()가 시트를 찾지 못합니다. 스프레드시트 만들기·코드 붙여넣기·웹앱 배포·URL 복사는 사용자마다 본인 구글 계정·본인 파일에서 각각 진행합니다.

    배포

    1. 연동한 스프레드시트에서 확장 프로그램 > Apps Script를 엽니다.
    2. 아래 코드를 붙여 넣고 저장합니다.
    3. 배포 > 새 배포 > 유형: 웹 앱, 실행 사용자: 나, 액세스: 누구나(또는 링크가 있는 사용자).
    4. 웹앱 URL(.../exec)을 복사해 대시보드에 입력합니다.
